# Engineering Mathematics: Terminal Velocity Derivation ## 1. Problem Definition How does the velocity of an object change as it falls through a fluid? We consider gravity and a velocity-dependent drag force. ## 2. Modeling (Newton's Second Law) The forces acting on the falling object are: * **Gravity (Downward)**: $F_g = mg$ * **Air Resistance (Upward)**: $F_d = cv$ According to Newton's Second Law ($F = ma$): $$m \frac{dv}{dt} = mg - cv$$ ## 3. Step-by-Step Derivation ### A. Separation of Variables Divide both sides by $(mg - cv)$ and multiply by $dt$: $$\frac{m}{mg - cv} dv = dt$$ ### B. Integration Integrate both sides from $t=0$ to $t$ (assuming $v(0)=0$): $$\int_{0}^{v} \frac{m}{mg - cv'} dv' = \int_{0}^{t} dt'$$ Using the substitution $u = mg - cv$, we get: $$-\frac{m}{c} \ln(mg - cv) \bigg|_{0}^{v} = t$$ ### C. Solve for Velocity $v(t)$ $$-\frac{m}{c} \left[ \ln(mg - cv) - \ln(mg) \right] = t$$ $$\ln\left( \frac{mg - cv}{mg} \right) = -\frac{c}{m}t$$ $$\frac{mg - cv}{mg} = e^{-\frac{c}{m}t}$$ Final expression: $$v(t) = \frac{mg}{c} \left( 1 - e^{-\frac{c}{m}t} \right)$$ --- ## 4. Key Properties * **Initial Acceleration**: At $t=0$, $a = g$. * **Terminal Velocity**: As $t \to \infty$, $v = v_T = \frac{mg}{c}$. * **Time Constant**: The rate of reaching terminal velocity depends on the ratio $m/c$.
