Engineering Mathematics: Terminal Velocity of a Falling Object

Problem

How does the velocity of an object change as it falls through air?

Modeling (Newton’s Second Law)

Forces acting on the object:

  • Gravity: mg
  • Air resistance (drag): cv

m dv/dt = mg - cv

Solving the Differential Equation

1. Rewrite

dv/dt = g - (c/m)v

2. Separate Variables

dv / (g - (c/m)v) = dt

3. Integrate

-(m/c) ln(g - (c/m)v) = t + C

4. Rearrange

g - (c/m)v = Ae^(-(c/m)t)

5. Solve for v(t)

v(t) = (mg/c)(1 - Ae^(-(c/m)t))

6. Initial Condition

v(0) = 0 → A = 1

Final Answer

v(t) = (mg/c)(1 - e^(-(c/m)t))

Physical Interpretation

  • As t → ∞, velocity approaches terminal velocity: v = mg/c
  • Initially accelerates
  • Acceleration decreases over time
  • Eventually reaches constant speed
